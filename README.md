# DevRoast 🔥

> Get your code roasted by an AI, climb the leaderboard of shame, and learn... maybe.

**DevRoast** is a fun, developer-centric platform where you can paste your code snippets to receive brutally honest (and hilarious) feedback. It's not just a code review; it's a roast session for your syntax errors and questionable logic.

Built during **Rocketseat's NLW (Next Level Week)** event. 🚀

## Features

- **The Roast:** Paste your code into our editor and let the AI roast it. Toggle "Roast Mode" for maximum sarcasm.
- **Leaderboard:** See the "worst" code snippets ranked by shame score. Compete for the bottom spot!
- **Sleek UI:** Dark mode by default, terminal-inspired aesthetics, and smooth animations.
- **Code Analysis:** Receive genuine (albeit snarky) insights into performance, security, and best practices.

## How to Run

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Technologies

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- **Biome** (Linting & Formatting)
- **Shiki** (Syntax Highlighting)
- **Base UI** (Headless Components)
